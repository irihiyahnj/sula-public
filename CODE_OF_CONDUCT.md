# Code Of Conduct

This project expects respectful, direct, and technically grounded collaboration.

## Our Standard

Participants should:

- focus on facts, code, and clear reasoning
- assume good intent while still challenging weak technical arguments
- keep feedback specific and actionable
- avoid harassment, intimidation, personal attacks, or discriminatory behavior

## Unacceptable Behavior

- personal abuse or threats
- sexualized language or unwanted attention
- publishing private information without consent
- repeated bad-faith disruption of project work

## Enforcement

Maintainers may remove or moderate content and participation that violates this standard.

## Project Context

Sula is an operating system for project collaboration. That means repo hygiene, reasoning quality, and rollout discipline are part of community behavior, not optional polish.
